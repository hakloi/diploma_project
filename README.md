# Final Project

The purpose of this website is to combine the openness of a blog platform like Medium and structured systematic learning as from Yandex Handbook, but with personalized progress. 

Project structure:
- /app - project core, includes settings, db and configuration data
- /models - includes db tables
- /schemas - pydantic schemas, validation models
- /api - endpoints for each entity
- /services - functions, project logic
- /utils - utilities functions
- requirements.txt - the list of required frameworks
- main.py - the start point of the project
